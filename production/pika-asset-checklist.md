# Pika Asset Checklist

Use this workflow for the first mobile launch:

1. Generate one reusable 10-second loop per task in Pika.
2. Download the MP4 without watermark if the selected Pika plan allows it.
3. Rename each clip to the exact task filename.
4. Put the files in `assets/videos/pika/`.
5. Reopen `http://localhost:4174/` and generate that task.
6. Confirm the Pika clip appears behind the captions.

Recommended first batch:

| Task | Filename | Scene goal |
| --- | --- | --- |
| Brushing teeth | `teeth.mp4` | Child refuses, then slowly accepts toothbrush and brushes. |
| Bath or shower | `bath.mp4` | Child refuses, then touches water and washes one arm. |
| Eating | `food.mp4` | Child refuses, then sits and takes one tiny bite. |
| Bedtime | `bedtime.mp4` | Child resists bed, then settles with blanket. |
| Getting dressed | `clothes.mp4` | Child refuses clothes, then puts on one sock or shirt. |
| Clean-up | `cleanup.mp4` | Child refuses, then puts one toy away. |
| Leaving | `leaving.mp4` | Child refuses transition, then takes three steps. |
| Car seat | `car.mp4` | Child resists, then sits and accepts buckle. |

Keep every clip visually consistent: same art style, same character proportions, same color palette, slow movement, and no hard cuts.
