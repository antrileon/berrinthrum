# Pika Video Assets

Place exported Pika MP4 files in this folder using these exact names:

- `teeth.mp4`
- `bedtime.mp4`
- `bath.mp4`
- `clothes.mp4`
- `cleanup.mp4`
- `leaving.mp4`
- `car.mp4`
- `food.mp4`

The app checks this folder when a video is generated. If the matching MP4 exists, the app shows the pre-rendered clip. If the file is missing, it keeps using the CSS cartoon fallback.

Optional poster images can go in `assets/posters/pika/` with matching names, for example `teeth.jpg`.
